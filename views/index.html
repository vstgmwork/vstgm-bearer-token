REST API

A REST API page developed using Node & Express

Accepted Parameters are listed below

  • /generate - This will generate a bearer token. This bearer token can be seen in the response section as JSON. Also the token will expire in 4 minutes. The time at which the token got generated will be shown in the headers section with the key utctime. This is just a API call and when opened in the browser will just show "Invalid Route".
  • /validate - This will accpect the bearer token as authorization field and will validate if the token is valid or not.
  • /delayload/second - This will load a html file after the mentioned time in seconds. http://localhost:3010/delayload/480 will load the page after 480 seconds
